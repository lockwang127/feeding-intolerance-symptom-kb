## 一、概述

### 1.1 什么是喂养不耐受（FI）

喂养不耐受（Feeding Intolerance, FI）是指胃肠道不能有效耐受肠内营养，表现为胃残余量过多、腹胀、呕吐、腹泻等症状，导致营养目标无法实现。FI是危重症患者、术后患者和早产儿营养支持中的常见挑战。

## 二、定义与诊断标准

### 2.1 成人FI定义

**主流定义（ESPEN/ASPEN）**：
- 胃残余量（GRV）> 500ml/6h 或 > 75% 输注量
- 伴有以下一项或多项：腹胀、恶心、呕吐、腹泻、误吸

**肿瘤患者FI定义**：
- 不能耐受肠内营养，出现腹胀、腹泻、恶心、呕吐、反流等症状，影响营养目标实现
- 住院肿瘤患者肠内营养不耐受发生率：**15-40%**

### 2.2 早产儿FI定义

**临床定义**：
- 生后14天仍不能达到150ml/kg/d的完全肠内喂养
- 伴有胃潴留、腹胀、呕吐等临床表现

**发生率**：
- 极低出生体重儿（ELBW <1000g）：**50-70%**
- 超低出生体重儿（<32周）：**30-50%**

### 2.3 鉴别诊断要点

| 疾病 | 关键鉴别点 |
|------|-----------|
| 坏死性小肠结肠炎（NEC） | 腹部X线肠壁积气、Bell分期 |
| 肠梗阻 | 腹部X线气液平、呕吐物性质 |
| 感染 | 发热、WBC/CRP/降钙素原 |
| 代谢性疾病 | 电解质紊乱、血糖异常 |

## 四、危险因素

### 4.1 成人危险因素

| 类别 | 具体因素 |
|------|----------|
| **患者因素** | 高龄（>65岁）、低白蛋白（<30g/L）、恶病质 |
| **疾病因素** | 腹水、腹腔感染、胃肠道肿瘤 |
| **治疗因素** | 化疗（CINV）、手术（全胃切除）、机械通气 |
| **营养因素** | 高渗配方、输注速率过快、喂养量过大 |

### 4.2 早产儿危险因素

| 因素 | 机制 |
|------|------|
| 早产（<32周） | 胃肠道发育不成熟 |
| 出生体重<1500g | 消化酶活性低 |
| 围产期窒息 | 肠道低灌注 |
| 配方奶喂养 | 缺乏母乳免疫因子 |
| 肠道菌群失调 | 双歧杆菌减少 |

## 五、干预手段

### 5.1 成人干预策略

#### 5.1.1 营养支持路径

```
肠内营养启动 → 评估耐受性 → 调整方案 → 达标维持
     ↓
  不耐受 → 降速/减量 → 加促动力药 → 更换途径
```

**启动时机**：
- 危重症/术后：24-48小时内启动
- 胃癌术后：24小时内肠内营养
- 肿瘤恶病质：根据PG-SGA评分

**能量目标**：
- 成人：25-30 kcal/kg/d
- 蛋白质：1.2-1.5 g/kg/d
- 老年外科：25 kcal/kg/d，蛋白1.2-1.5g/kg/d

#### 5.1.2 促动力药物

| 药物 | 剂量 | 机制 |
|------|------|------|
| 甲氧氯普胺 | 10mg q6-8h iv/po | 多巴胺受体拮抗剂 |
| 红霉素 | 3-5mg/kg/d ×5-7d | 胃动素受体激动剂 |
| 多潘立酮 | 10mg q8h po | 外周多巴胺拮抗剂 |

#### 5.1.3 管饲途径选择

| 途径 | 适应证 |
|------|--------|
| 鼻胃管 | 短期（<4周）、胃排空正常 |
| 鼻空肠管 | 胃排空障碍、高误吸风险 |
| 胃造口 | 长期（>4周）口服不足 |
| 空肠造口（PEJ） | 胃癌术后、胰腺手术 |

### 5.2 早产儿干预策略

#### 5.2.1 非药物干预

| 方法 | 证据等级 | 推荐 |
|------|----------|------|
| **微量喂养** | I级 | 0.5-1ml/kg/d起始，每3-5天递增15-20ml/kg |
| **非营养性吸吮** | I级 | 管饲前30分钟安抚奶嘴 |
| **腹部按摩** | I级 | 顺时针5-10分钟，每日2-3次 |
| **鸟巢式护理** | I级 | 模拟子宫环境 |
| **右侧卧位** | I级专家共识 | 喂养后30分钟 |

#### 5.2.2 益生菌干预

**NHS UK 2025推荐**：
- 适用：<32周或<1500g早产儿
- 方案：BB-02 + LGG + 布拉氏酵母菌三联
- 剂量：每日1次，每次10亿CFU
- 疗程：生后24h开始至矫正胎龄36周或出院

**效果**：缩短全肠内营养时间3-5天，减少FI发生率30%

### 5.3 肿瘤患者特殊考量

#### 5.3.1 化疗相关恶心呕吐（CINV）

| 风险等级 | 致吐风险 | 推荐方案 |
|----------|----------|----------|
| 高危 | >90% | 5-HT3 + NK1 + 地塞米松 |
| 中危 | 30-90% | 5-HT3 + 地塞米松 |
| 低危 | 10-30% | 地塞米松单药或按需 |

#### 5.3.2 恶病质营养干预

- 高蛋白、高能量密度（1.5-2.0 kcal/ml）
- 联合食欲刺激剂（甲地孕酮、甲羟孕酮）
- ω-3脂肪酸补充

## 六、干预后效果评估

### 6.1 主要评估指标

| 指标 | 成人 | 早产儿 |
|------|------|--------|
| **主要** | 达目标能量时间、住院时间 | 达全肠内营养时间（150ml/kg/d） |
| **次要** | 并发症发生率、体重变化 | 体重增长速率、并发症 |
| **长期** | 生存率、生活质量 | 神经发育、追赶生长 |

### 6.2 效果预期

| 干预 | 成人效果 | 早产儿效果 |
|------|----------|------------|
| 益生菌 | - | 缩短全肠内营养3-5天 |
| 早期肠内营养 | 缩短住院2-3天 | 降低NEC风险50% |
| 非药物干预 | - | 缩短住院5-7天 |
| 综合干预 | 80-90%康复率 | 85%在7日内建立耐受 |

### 6.3 康复判断标准

**成人**：连续3天无呕吐、腹胀，GRV<30%

**早产儿**：
- 连续3天无呕吐、腹胀
- 胃残余量<30%前次喂养量
- 可耐受喂养递增

## 十、结论

喂养不耐受（FI）是临床营养支持中的重要挑战，涉及成人重症/肿瘤/外科和新生儿/早产儿两大核心人群。本知识库系统整合了2024-2026年最新指南共识和临床证据，包含400条结构化知识三元组，覆盖定义诊断、评估工具、危险因素、干预策略和效果评价全流程。

**核心要点**：
1. **早期识别**：NRS 2002/PG-SGA筛查 + FI评分系统
2. **阶梯干预**：非药物优先，药物促动力，必要时更换途径
3. **多学科协作**：营养科、ICU、外科、儿科共同参与
4. **持续监测**：闭环管理，定期评估调整
